#  Valentine’s Day Spending Analysis Dashboard

##  Project Overview

This project analyzes **Valentine’s Day spending behavior** using **Power BI**. The dashboard provides insights into consumer spending patterns, sentiment scores, gift preferences, and demographic trends.

The goal of this project is to demonstrate **data visualization, business intelligence, and dashboard design skills**.

---

## 🛠 Tools & Technologies

* Microsoft Power BI
* Data Visualization
* Data Analysis
* Dashboard Design

---

##  Dataset

The dataset contains information about:

* User demographics
* Spending amount
* Gift types
* Relationship status
* Sentiment scores
* City-wise spending

---

##  Dashboard Insights

The dashboard includes the following visualizations:

* **KPI Cards**

  * Total Spending
  * Total Users
  * Average Spending
  * % Celebrating Valentine’s Day

* **Spending by Gender**

* **Spending by Gift Type**

* **Spending by Age Group**

* **Relationship Status vs Spending**

* **City-wise Spending (Map Visualization)**

* **Spending vs Sentiment Score (Scatter Plot)**



##  Key Insights

* Identifies which **cities spend the most on Valentine’s Day**
* Shows **spending differences by gender**
* Analyzes **sentiment score vs spending behavior**
* Highlights **popular gift categories**
